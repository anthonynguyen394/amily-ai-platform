# Workflows

Exported n8n workflow JSON files. Import these into your n8n instance via the UI or API.

## Naming Convention

`[category]-[description].json`

Examples:
- `voice-post-call-processing.json`
- `review-request-sms.json`
- `onboarding-client-setup.json`

## Importing Workflows

1. Open n8n at `https://n8n.amily.ai`
2. Go to **Workflows** > **Import from File**
3. Select the JSON file
4. Update any credential references to match your n8n instance
5. Activate the workflow
